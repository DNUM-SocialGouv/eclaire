# API Eclaire - Plateforme nationale des essais cliniques - Ministère de la santé
## Installation

- Dependencies
```bash
$ npm install
```

- Duplicate `.env.dist` into `.env`, then update db credentials

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
